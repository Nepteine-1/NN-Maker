# NN Maker

Software for creating and training neural networks 

### Ubuntu package
- Qt5
```bash
sudo apt install qttools5-dev libqt5charts5-dev qtbase5-dev qtchooser qt5-qmake qtbase5-dev-tools
```
- CMake
```bash
sudo apt install cmake
```
- g++ 
```bash
sudo apt install g++
```
